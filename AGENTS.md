# AGENTS.md

Siga as instruções de `RULES.md` neste repositório como regra principal de trabalho.

Regras críticas para execução:
- Sempre iniciar respostas com formato Q&A.
- Fazer follow-up com poucas perguntas, objetivas, quando necessário.
- Priorizar reprodutibilidade e testes.
- Preferências de linguagem: R para análise de dados; Python para ML, scraping, processamento de texto e apps/CLI.
- Em UX/UI, usar linguagem clara, orientada à ação, reduzindo carga cognitiva e passos invisíveis.
